# 📊 Database Design and Administration using MySQL

A comprehensive **Database Design and Administration** project developed using **MySQL**. This repository demonstrates the design, implementation, and administration of three real-world database systems: **Bank Management System**, **Hospital Management System**, and **Student Result Management System**.

The project covers database schema design, relational modeling, SQL queries, database security, user management, views, indexes, stored procedures, triggers, and role-based access control.

---

## 🚀 Project Objectives

* Design normalized relational databases for real-world applications.
* Implement SQL queries for data management and reporting.
* Create secure database users with role-based permissions.
* Automate database operations using stored procedures and triggers.
* Optimize database performance using indexes and views.

# ▶️ How to Run

1. Install MySQL Server and MySQL Workbench.
2. Clone this repository.
3. Open the SQL scripts in MySQL Workbench.
4. Execute the schema creation scripts.
5. Insert sample data.
6. Create users and assign permissions.
7. Execute views, indexes, stored procedures, and triggers.
8. Test the project using both Developer and Tester users.
